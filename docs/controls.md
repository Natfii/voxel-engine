# Controls

## Movement

- **W** - Move forward
- **S** - Move backward
- **A** - Strafe left
- **D** - Strafe right
- **Space** - Jump (or swim up in water)
- **Left Shift** - Sprint (hold) / Swim down in water
- **Mouse** - Look around

## Special Modes

- **N** - Toggle noclip mode (fly through walls)
- **I** - Open/close inventory
- **F9** - Open/close developer console
- **ESC** - Open pause menu / Close console / Close inventory

## Noclip Mode

When noclip is enabled:
- **W/S/A/D** - Move horizontally
- **Space** - Fly up
- **Left Shift** - Fly down
- No collision detection
- No gravity

## Interaction

- **Left Mouse** - Break block
- **Right Mouse** - Place block or structure (creative mode)
- **0-9 Number Keys** - Select items from hotbar (when inventory is open)

## Console

Press **F9** to open the console and type commands.
See `docs/console.md` for console usage.

## Configuration

Edit `config.ini` to customize key bindings and settings.
